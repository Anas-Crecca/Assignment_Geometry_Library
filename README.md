## About The Project
A C# program that creates a few randomized 3D shapes and calculates their Surface-Area, Volume and an assorted collection of other things, as well as adding said shapes together into CompoundShapes, calculating their compound messurments, sorting them and checking them for certain shapes.

## Getting Started 
-   Download & Extract Zip Folder
- Open Project in IDE of Choice (f.e. Visual Studio Code)
- Change diretory via terminal: 
    - cd Computation
- Run the program using: 
    - dotnet run

### Prerequisites
- Have .NET 10 installed

### Installation
- Download & Extract Zip Folder

### Usage
- if you need some randomised Tetrahedons, Cuboids or Cylinders and their Surface_Area and Volume
- as a base for 3D shape calculation for your own work
- if you need to calculate the Volume of a Tetrahedon/Cuboid/Cylinder and have a lot of patients to keep running the program till it randomizes the correct Coordinates for your Tetrahedon/Cuboid/Cylinder (not recommended)
- might also get you out of a saw trap if said trap revolves around knowing how to calculate stuff related to 3D shapes... (also not recommended)
